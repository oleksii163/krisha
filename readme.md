# Кровля Профі Planner

Легкий веб-додаток без залежностей, що запускається напряму з `index.html` та допомагає створювати 2D-плани дахів, формувати параметризований JSON та експортувати креслення у PDF і табличні дані у Excel.

## Можливості

- Майстер із трьох кроків: вибір типу даху, базові параметри, графічний редактор.
- 2D-редактор на SVG з підтримкою додавання та перетягування вузлів, вставки отворів, snap до сітки.
- Каталог шаблонів дахів (одно-, дво-, чотирьохскатні, ламані, Г-/Т-подібні, довільні).
- Панель параметрів із розрахунковими величинами площин, eaves, матеріалом, нотатками.
- Автовалідація моделі (перетини, площа, розміри) з блокуванням експорту при помилках.
- Автозбереження у LocalStorage кожні 30 секунд і при кожній зміні.
- Експорт у JSON, PDF (векторний план з легендою) та Excel (.xlsx) без зовнішніх бібліотек.

## Як запустити

1. Відкрийте `index.html` у сучасному браузері (Chrome, Edge, Firefox).
2. Дотримуйтесь підказок майстра, щоб сформувати модель.
3. Перейдіть на крок «Редактор», щоб дорисувати або відкоригувати план.
4. Кнопка «Експорт» активна після усунення всіх помилок валідації.

## Структура проекту

```
index.html      – основна розмітка
styles.css      – темна тема, сітка, панелі керування
src/
  main.js       – ініціалізація та керування кроками
  state/        – стан редактора та шаблони дахів
  geometry/     – обчислення площ, периметрів, валідація
  ui/           – компоненти інтерфейсу (редактор, панелі)
  export/       – генератори JSON, PDF та XLSX
  utils/        – допоміжні утиліти (збереження, ZIP)
```

## Відомі обмеження

- 2D-редактор підтримує одну замкнену оболонку для площини; складні топології потребують ручного доопрацювання.
- PDF містить базовий кресленик з підписами площин без детальної специфікації ребер.
- Excel генерується в UTF-8 із мінімальним стилюванням.

Будь-які покращення або оптимізації вітаються через pull request.
